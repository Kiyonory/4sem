Кириллица в PDF (ReportLab)
============================

Если в скачанном PDF русские буквы отображаются квадратами, положите сюда файл:

  DejaVuSans.ttf

Скачать (свободная лицензия): https://dejavu-fonts.github.io/Download.html
Нужен файл из архива: ttf/DejaVuSans.ttf

После этого генерация PDF в админке подхватит шрифт на любой ОС.

На многих Mac также сработает системный Arial Unicode без этого файла
(путь проверяется в shop/pdf_fonts.py).
